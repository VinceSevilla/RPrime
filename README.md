# RPrime 

RPrime is a modern web-based hotel booking system designed to provide a seamless and elegant experience for guests. Users can browse luxury rooms, restaurants, and events, and book their stay with an intuitive interface

## Features
- User authentication with Supabase (Email/Password & Google OAuth)
- Room browsing with detailed descriptions, prices, and images
- Booking system with check-in/check-out date selection
- Restaurant and event listings
- Responsive UI styled with Tailwind CSS

## Tech Stack
- **Frontend:** React + Vite
- **Framework:** Tailwind CSS
- **Database:** Supabase
